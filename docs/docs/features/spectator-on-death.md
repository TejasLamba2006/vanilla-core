# Spectator On Death

Moves players into spectator mode right after they respawn.

## Configuration

```yaml
features:
  spectator-on-death:
    enabled: false
```

## In-Game Usage

- Open `/vanilla` menu
- Find `Spectator On Death`
- Left-click to enable or disable

When enabled, players automatically switch to spectator mode after death.
